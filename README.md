# nestjs-api-example

## Overview

- This is CRUD Api basic.
- Main concept:
  - Using docker
  - CRUD api
  - Using swagger
  - Using jwt-auth token
  - Send mail
  - Working with localstack

## Installed

- Swagger:
  ```
    npm install --save @nestjs/swagger swagger-ui-express
  ```
- TypeOrm:
  - Document : https://typeorm.io/
  ```
    npm install --save typeorm mysql2 @nestjs/typeorm
  ```
